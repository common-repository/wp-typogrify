== Description ==

wp-Typogrify is dead.  wp-Typogrify has merged with wp-Hyphenate to become [wp-Typography](http://wordpress.org/extend/plugins/wp-typography/), an all-inclusive typography solution. Long live wp-Typogrify!

If you have this plugin installed, please uninstall it and install [wp-Typography](http://wordpress.org/extend/plugins/wp-typography/).

A special thank you is due to [Hamish Macpherson](http://blog.hamstu.com/) who is responsible for the success of wp-Typogrify, and without whom, wp-Typography would not have been possible.

Do *not* install this plugin.  This plugin has been replaced by [wp-Typography](http://wordpress.org/extend/plugins/wp-typography/).
